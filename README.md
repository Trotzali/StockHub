# StockHub

ASX stock screening platform. Solo dev project.

Daily end-of-day signals over the ASX 200, with potential expansion to
a separate crypto/shitcoin section once the equities engine is validated.

Stack: Python 3.12 / Streamlit / Supabase / Plotly / yfinance.

Status: bootstrapping. See `_project_state.md` for current scope and
open work packages, `_build_log.md` for shipped commits.
